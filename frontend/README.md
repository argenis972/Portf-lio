# Frontend - Portfólio Pessoal

Frontend do portfólio pessoal de Argenis Lopez, desenvolvido com React, TypeScript e Tailwind CSS.

## Tecnologias
- React 18
- TypeScript
- Vite
- Tailwind CSS

## Como rodar

```bash
cd frontend
npm install
npm run dev
```

O frontend roda em http://localhost:5173 e precisa do backend rodando em http://localhost:8000.

## Funcionalidades
- Tema claro/escuro
- Internacionalização (PT-BR, ES, EN)
- Formulário de contato via Formspree
- Design responsivo

## Nota sobre foto de perfil

Substitua `public/perfil.jpg` com sua foto real (recomendado: imagem quadrada, pelo menos 400x400px). 
Se a foto não existir, será exibido um fallback com as iniciais "AL".
